## Step by Step guide

### Step-1: Check the initial status
- ```uv run ruff check .``` : this checks all the files in the project

### Step-2: Apply initial auto fixes
- ```uv run ruff check . --fix```: Auto fixing the error

### Step-3: Check for format error
- ```uv run ruff format .```

- ```uv run ruff check .```
